## Platform Goals

### 1. From-The-Ground-Up Implementation

*   Vector operations
*   Matrix operations
*   Back and forward propagation
*   Logistic regression
*   Gradient descent
*   Loss functions
*   Activation functions
*   Neural networks
*   CNNs
*   RNNs
*   Regularization
*   Graph networks


### 2. Compatibility

*   Runs on Node.js
*   Runs on modern browsers


### 3. Speed

Decidedly unoptimized -- written for convenience, not for speed.
